*AUDIBLE Audiobooks Data Analysis*


*PROJECT OVERVIEW*

This project analyses audiobook data using Power BI to uncover insights related to pricing, ratings, duration and content distribution. The data was preprocessed to ensure accuracy, and interactive visualizations were created to explore key patterns and relationships. The aim of the project is to transform raw data into meaningful insights and present them through an intuitive dashboard for better understanding and decision-making.


*OBJECTIVES*

•	Analyze audiobook pricing patterns
•	Study rating distribution
•	Identify top authors and books
•	Examine relationship between duration and price
•	Build an interactive dashboard


*TOOLS USED*

•	Excel
•	Power Query
•	Power BI


*STEPS FOLLOWED*

•	Cleaned and preprocessed the dataset
•	Transformed columns (rating, time, etc.)
•	Converted time into numerical format (minutes)
•	Built multiple visualizations
•	Created interactive dashboards using slicers and filters


*KEY INSIGHTS*

•	Most audiobooks have ratings above 4.5
•	Longer audiobooks tend to be priced higher
•	Some short audiobooks are premium priced
•	Certain authors dominate the dataset
•	English is the most common language


*DASHBOARD FEATURES*

•	KPI cards for quick insights
•	Bar and column charts for comparison
•	Donut chart for rating distribution
•	Scatter plot for duration vs price analysis
•	Interactive slicers for filtering


*CONCLUSION*

The project demonstrates how data visualization can transform raw data into meaningful insights, helping in better understanding of audiobook trends.
